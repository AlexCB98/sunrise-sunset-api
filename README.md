# Sunrise & Sunset API

A simple Python project that retrieves sunrise and sunset times for a specific location using a public API.

This project was built using Python and the `requests` library.

---

## Features

* Retrieve sunrise and sunset times from an API
* Display the results in the terminal

---

## What I practiced

* Working with APIs
* Sending HTTP GET requests
* Using query parameters
* Reading JSON data
* Extracting nested dictionary values

---

``` 
main.py

```

## How to run

Run the project with:

```bash

python main.py

```
